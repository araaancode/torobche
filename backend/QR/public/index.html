API Usage

You can also use our API to generate QR codes programmatically:

// Generate QR code as image
GET /generate?text=YourTextHere

// Generate with custom size
GET /generate-custom?text=YourText&size=400

// Download QR code
GET /download?text=YourText&filename=myqrcode

// API endpoint (returns base64)
POST /api/qrcode
{ "text": "YourText", "format": "png" }
